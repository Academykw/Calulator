# Joe Annel Calculator

A sleek, powerful, and intuitive scientific calculator built with Flutter.

## Overview

Joe Annel Calculator is designed to provide a seamless mathematical experience for students and professionals alike. It combines basic arithmetic with advanced scientific functions in a clean, modern interface that supports both light and dark themes.

## Features

- **Advanced Mathematics:** Support for trigonometry (sin, cos, tan), logarithms (log, ln), square roots, and exponents.
- **Mathematical Constants:** Quick access to Pi (π) and Euler's number (e).
- **Dynamic UI:** 
    - Real-time expression evaluation.
    - Light and Dark mode support with a dedicated toggle.
    - Scrollable display for long expressions and results.
- **State Management:** Powered by `Provider` for a responsive and efficient user experience.

## Getting Started

This project is a functional Flutter application. To run it locally:

1.  **Prerequisites:** Ensure you have the [Flutter SDK](https://docs.flutter.dev/get-started/install) installed.
2.  **Clone the repo:**
    ```bash
    git clone https://github.com/Academykw/Calulator.git
    ```
3.  **Install dependencies:**
    ```bash
    flutter pub get
    ```
4.  **Run the app:**
    ```bash
    flutter run
    ```

## Project Structure

- `lib/providers/`: Contains `CalculatorProvider` for logic and `ThemeProvider` for UI state.
- `lib/screens/`: UI layouts including the main calculator and splash screen.
- `lib/widgets/`: Reusable UI components like `CalculatorButton`.
